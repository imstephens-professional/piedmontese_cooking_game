## 1.0 Description
A game that resembles Cooking Mama. 

## 2.0 Requirements
* Main Menu (Prompts the user to Start Cooking or to Quit Game).
* A Recipe Menu (Keeps track of Recipes Unlocked, Recipes Not Unlocked, and the Score Achieved for Completing a Recipe).
* Recipes (Which contains a list of Mini-Games)
* Mini-Games (Which requires input from the user, a timer, and a score)
